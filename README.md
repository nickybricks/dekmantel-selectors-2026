# Dekmantel Selectors 2026 — Line-up Explorer

Interaktiver Cover-Flow-Explorer für das Dekmantel Selectors 2026 Line-up (20.–24. August, The Garden Resort, Tisno, Kroatien).

## Features
- Cover Flow im iTunes-Stil: 70 Acts, per Mausrad, Ziehen oder Touch-Wischen durchblättern
- Echtes Artist-Bild pro Act, live über die oEmbed-Schnittstellen geladen
- 83 einzeln hörbare DJs, 81 davon mit Player — bei B2B-Sets ist jeder DJ über seinen Namen im Titel wählbar
- Spotify und SoundCloud als Tabs, wo beides vorhanden ist
- Nicht abspielbare SoundCloud-Mixe werden als solche ausgewiesen statt eines toten Players
- Suche + Genre-Filter (House, Techno, Disco, Bass/Club, Jungle/D&B, Eclectic)
- Merkliste per Herz-Button, bleibt über Browser-Neustarts erhalten
- Mobil-tauglich (iPhone/Android)

## Nutzung
Statische Seite, keine Dependencies, kein Build. `index.html` im Browser öffnen oder auf GitHub Pages hosten.

## Hinweis
Inoffizielles Fan-Projekt — weder von Dekmantel Selectors betrieben noch autorisiert. Das Line-up ist ohne Gewähr; verbindlich ist allein [dekmantelselectors.com](https://dekmantelselectors.com).

Artist-Bilder und Player laden live über die offiziellen oEmbed-Schnittstellen von SoundCloud/Spotify.
